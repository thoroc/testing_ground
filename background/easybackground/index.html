Easy background

(source)

With the easy backgrounds jQuery plugin you can add animated and non animated backgrounds to your webpage or any HTML element with only a few lines of code!

The plugin is just 14kb and supports particles, slideshows, videos, gradients and animated patterns.

You can customize your background using 36+ options and some preset overlays or you can use a image as overlay!
